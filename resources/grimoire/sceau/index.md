# SEO Management with Sceau

Sceau gives you a full suite of SEO tools built directly into your admin panel. For every piece of content — pages, products, blog posts, or any other model — you can manage exactly how it appears in search engines, social media previews, and AI-powered search results.

## Where to find SEO settings

When editing a record that has Sceau enabled, scroll down below the main form to find the **SEO** panel.

![The SEO relation manager on a resource edit page](/grimoire-asset/sceau/seo-relation-manager.png)

If no SEO data has been added yet, you'll see an empty state with an **Add SEO Data** button. Click it to open the SEO editor.

## The SEO editor

The SEO editor is organised into five tabs, each covering a different aspect of search and social optimisation:

| Tab | Purpose |
|-----|---------|
| **Basic SEO** | Core meta tags for search engines |
| **Open Graph** | How content appears on Facebook, LinkedIn, etc. |
| **Twitter Card** | How content appears on Twitter / X |
| **Schema Markup** | Structured data (JSON-LD) for rich results |
| **AI Optimization** | Content freshness signals for AI search engines |

Most fields are optional with sensible defaults and fallback chains — you don't have to fill in everything. Start with Basic SEO and work outward as needed.

## Global settings

Site-wide SEO configuration (your business name, address, contact details, and opening hours) is managed separately under **Settings → SEO Settings** in the sidebar.
